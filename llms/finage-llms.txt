# Finage

> Finage is a real-time and historical market data provider covering US and global stocks, forex, cryptocurrencies, CFD indices, ETFs, bonds, and company fundamentals — delivered over a REST API (api.finage.co.uk, apikey query-parameter auth, all endpoints GET, gzip required) and dedicated WebSocket streams (per-customer subdomain on finage.ws, socket-key token auth). Sign up at https://moon.finage.co.uk/register; a 3-day free trial covers all markets.

Generated by API Evangelist from the Finage docs database (https://api.finage.co.uk/op/v1/endpoints/list) and the artifacts in https://github.com/api-evangelist/finage. Finage does not publish its own llms.txt (checked https://finage.co.uk/llms.txt on 2026-07-22).

## APIs

- [Finage Market Data API](https://finage.co.uk/docs/api): 72 GET endpoints across US Stocks, Forex, Crypto, Indices, ETFs, UK/Indian/Canadian/Russian stocks, Bonds, and Fundamentals (statements, calendars, news, technical indicators, search)
- [Finage WebSocket streams](https://finage.co.uk/docs/websocket): real-time trades and quotes; your assigned wss URL, port, and socket key are in the dashboard
- [Symbol lists](https://finage.co.uk/docs/symbols): symbology reference for all covered markets

## Key conventions

- Authenticate every REST request with `?apikey=YOUR_API_KEY`
- Send `Accept-Encoding: gzip` — the API returns 406 `{"error":"GZip compression required"}` without it
- Errors come as a single JSON envelope: `{"error":"<message>"}`
- Timestamps are Unix epoch milliseconds; date range parameters use YYYY-MM-DD
- WebSocket errors: `401 - invalid socket key`, `401 - use your socket key to connect`, `401 - please upgrade your package to access this market`
